# Campus Recruitment Data Science Project

## Deskripsi
Proyek ini bertujuan untuk menganalisis faktor akademik dan employability yang mempengaruhi keberhasilan mahasiswa dalam program Campus Recruitment menggunakan pendekatan Machine Learning.

## Tujuan Bisnis
- Meningkatkan tingkat keberhasilan placement mahasiswa
- Mengidentifikasi faktor yang paling berpengaruh terhadap placement
- Mendukung pengambilan keputusan akademik berbasis data

## Model yang Digunakan
- Logistic Regression
- Random Forest Classifier

Model terbaik: Random Forest dengan akurasi 81%

## Faktor Paling Berpengaruh
1. Nilai rata-rata SMP
2. IPK
3. Nilai rata-rata SMA
4. Nilai pascasarjana
5. Nilai tes kemampuan kerja

## Insight
Prestasi akademik dan tes kemampuan kerja memiliki pengaruh signifikan terhadap keberhasilan placement mahasiswa.

# Campus Recruitment Data Science Project

## Deskripsi Projek
Proyek ini bertujuan untuk menganalisis faktor akademik dan employability yang mempengaruhi keberhasilan mahasiswa dalam program Campus Recruitment menggunakan pendekatan Machine Learning. Analisis dilakukan untuk membangun model prediksi status placement serta mengidentifikasi variabel yang paling berpengaruh terhadap keberhasilan mahasiswa mendapatkan pekerjaan.

Dataset yang digunakan berisi data historis mahasiswa meliputi latar belakang pendidikan, IPK, pengalaman kerja, serta nilai tes kemampuan kerja.

## Tujuan Bisnis
- Meningkatkan tingkat keberhasilan placement mahasiswa
- Mengidentifikasi faktor yang paling berpengaruh terhadap placement
- Mendukung pengambilan keputusan akademik berbasis data

## Metodologi
Tahapan yang dilakukan dalam proyek ini meliputi:
-Data Understanding & Validation
-Data Cleaning & Preprocessing
-Encoding variabel kategorikal
-Train-Test Split (80:20)
-Model Building
-Model Evaluation
-Feature Importance Analysis

## Model yang Digunakan
- Logistic Regression
- Random Forest Classifier

Model terbaik: Random Forest dengan akurasi 81%

## Faktor Paling Berpengaruh
1. Nilai rata-rata SMP
2. IPK
3. Nilai rata-rata SMA
4. Nilai pascasarjana
5. Nilai tes kemampuan kerja

## Insight
Hasil analisis menunjukkan bahwa konsistensi prestasi akademik sejak jenjang awal pendidikan serta performa pada tes kemampuan kerja memiliki pengaruh signifikan terhadap keberhasilan placement mahasiswa. Model prediksi yang dibangun dapat digunakan sebagai sistem pendukung keputusan dalam evaluasi kesiapan kerja mahasiswa.
